Usage Case

A example device of X-ray scanner, Siemens' mobile C-arms system is for X ray scanning and image processing.

In surgery, it is crucial to keep the Aseptic environment. Currently, there is a trend in adapting posture interaction to control the device. This project is a potential solution applied in image processing.
